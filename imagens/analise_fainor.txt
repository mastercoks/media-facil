Site da Fainor

- O site não aprensenta layout responsivo, principalmente para acesso em dispositivos mobile, dificultando a visualização do conteúdo em tais dispositivos.

-Apresenta opção de alto contraste, funciona bem mas, oculta as imagens do site, deixando visivel apenas os textos.

-Apresente opção para aumentar e reduzir a fonte do site, quando ativo o layout do site se quebra, dificultando a visualização das informações

-Utilização de icones relacionados as informações e opções no menu

-Ao passar a página pricipal do site no W3C Validator, apresentou 4 erros nos elementos <frameset>

-As páginas apresentam elementos com bastante interação com o mouse.

-Muita utilização de frameset (antigo iframe) no desenvolvimento do site

-Página principal com informações faceis de encontrar, porém o caminho para se chegar realmente onde quer ainda depende de muitos clicks

-A validação do site no validador de CSS do W3C passou sem nenhum erro de CSS

-Imagens como background sem utilizar a tag para referenciar imagem
